## 5. LLM Wiki Operating Model (Karpathy Pattern)

The internal memory model follows a deterministic loop inspired by the Karpathy LLM Wiki gist:

1. Raw sources are gathered into structured JSON files.
2. A generated wiki index consolidates source metadata and totals.
3. A JSON schema validates the wiki-index contract.
4. A JSONL log records every regeneration event for auditability.
5. Public-facing docs consume generated index files instead of hand-edited fragments.
